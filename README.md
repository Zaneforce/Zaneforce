### What I Do at CoLearn

Building and maintaining production services for an edtech platform serving students across Indonesia:

- **Frontend** -- Admin dashboards (React), marketing site (Next.js), finance tools (React + Vite)
- **Mobile** -- Android app (Kotlin, MVVM) and cross-platform app (React Native + Expo)
- **Backend** -- Payment services, notification systems, live classes, URL shortener (Rails, Go, Express)
- **AI** -- Serverless AI features powered by Google Gemini and Cloud TTS
- **Reliability** -- Internal dashboard for monitoring system health (Next.js 15, New Relic)
